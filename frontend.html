FrontEnd Development

HTML - Lenguaje de Marcado de Hipertexto

HTML es la base de cualquier página web. Se utiliza para estructurar contenido en la web. Cada elemento de HTML es un bloque de construcción de la página, como títulos, párrafos y contenedores.

Ejemplo básico de HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Página de Ejemplo</title>
</head>
<body>
    <h1>Este es un título</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

CSS - Hojas de Estilo en Cascada

CSS se utiliza para controlar la presentación, el formato y el diseño de los documentos HTML. CSS permite separar el contenido de la página de su diseño visual, lo que puede mejorar la accesibilidad y la flexibilidad.

Ejemplo básico de CSS:

body {
    font-family: 'Arial', sans-serif;
    background-color: #f4f4f4;
}

h1 {
    color: #333;
}

JavaScript - Lenguaje de Programación del lado del cliente

JavaScript se usa para crear contenido interactivo en sitios web. Permite a los desarrolladores implementar funciones complejas y dinámicas, desde cargar contenido sin recargar la página hasta crear gráficos 2D y 3D complejos.

Ejemplo básico de JavaScript:

document.getElementById('demo').innerHTML = '¡Hola, Mundo!';
